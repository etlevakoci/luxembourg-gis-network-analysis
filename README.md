# GIS Proximity & Network Analysis: Luxembourg 🇱🇺

This repository hosts a complete ArcGIS Desktop project bundled as a **Map Package (`.mpk`)** focusing on **Proximity and Network Analysis** within the Grand Duchy of Luxembourg. The analysis models real-world transportation networks to evaluate urban accessibility, service catchments, and optimal routing infrastructure across the country.

---

## 📊 Core Analyses Performed

* **Isochrone / Service Area Mapping:** Generated 5, 10, and 15-minute travel time catchments from core urban hubs (Luxembourg City, Esch-sur-Alzette, Ettelbruck) to evaluate infrastructure reach.
* **Closest Facility Routing:** Solved complex network matrices to determine the most efficient emergency or service routes to critical infrastructure.
* **Accessibility Assessment:** Quantified coverage gaps by overlaying local demographic layers with service travel buffers.
